# youth-flow-attend

Simple attendance app for in-person meetings.

Quick start

1. Install dependencies

```powershell
npm install
```

2. Run the dev server

```powershell
npm run dev
```

Deployment

Deploy to Vercel or another static host. Configure the Vite env variables (VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY) in your hosting provider's environment settings.

Tech stack

- Vite + React + TypeScript
- Tailwind CSS
- Supabase (backend)

This README was simplified to remove third-party generator references.
